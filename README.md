# Retail Sales Intelligence Dashboard

## Overview

This project is an interactive Power BI dashboard designed to analyze retail sales data and extract meaningful business insights. It focuses on understanding sales performance across time, regions, and product categories.

## Dataset

The analysis is based on two datasets:

* **Orders**: Contains order-level information such as date, region, and order ID
* **Details**: Includes product-level data like category, quantity, and sales amount

These datasets are connected using a common key to enable integrated analysis.

## Key Metrics

The following KPIs are calculated using DAX:

* Total Revenue
* Total Orders
* Average Order Value
* Total Quantity Sold

## Dashboard Features

* Monthly sales trend analysis to identify seasonality
* Category and sub-category performance comparison
* Region-wise sales distribution
* Top-performing products identification
* Interactive filters for dynamic exploration

## Key Insights

* Sales show clear seasonal patterns with peak periods
* A few categories contribute a large portion of total revenue
* Certain regions underperform despite consistent order volume
* Top products drive a significant share of overall sales

## Tools & Techniques

* Microsoft Power BI (data visualization)
* DAX (calculated measures)
* Data modeling (table relationships)

## Outcome

This project demonstrates how raw sales data can be transformed into actionable insights to support better business decisions.
